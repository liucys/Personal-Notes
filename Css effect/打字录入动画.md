#### CSS 实现动画打字录入效果

> html 布局

```html
<h1>Pure CSS Typing animation.</h1>
```

> css 样式实现

```css
h1 {
  width: 16.5em;
  width: 26ch;
  margin: 2em 1em;
  /* 当不设置边框颜色值时，会使用字体的颜色作为边框的颜色 */
  border-right: solid 1px;
  /* 不自动换行 */
  white-space: nowrap;
  /* 溢出隐藏 */
  overflow: hidden;
  /* 动画 */
  animation: spac 3s steps(26, end), caret 0.3s step-end infinite alternate;
}

@keyframes spac {
  0% {
    width: 0;
  }
}

@keyframes caret {
  50% {
    border-color: transparent;
  }
}
```
